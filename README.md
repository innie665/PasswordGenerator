# Password Generator
A simple password generator built by using HTML,CSS and Javascript.It creates secure,random passwords.

##Features
-Generates random passwords
-Includes uppercase letters
-Includes lowercase letters
-Includes numbers
-Copy password to clipboard


##Technology used
-HTML
-CSS
-Javascript

##How to use
1.Open 'index.html' in your browser.
2.Select password options.
3.Click **Generate Password**
4.Copy and use as your next password.
